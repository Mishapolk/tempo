# TEMPO — Time Perception Game

A fast, elegant browser game that tests and trains your internal sense of time.

## Play it live
**[mishapolk.github.io/tempo](https://mishapolk.github.io/tempo)**

## Modes

| # | Mode | Description |
|---|------|-------------|
| 01 | **Estimate** | An interval runs for a random time. Count in your head, then type how long it lasted. |
| 02 | **Hold** | A target duration is shown. Hold the button for exactly that long. |
| 03 | **Mirror** | Watch an interval, then hold the button for the same duration. |
| 04 | **Blind Hold** | Like Hold, but all visual feedback is hidden. Pure instinct. |
| 05 | **Rhythm** | A metronome plays at a hidden BPM. Tap along to match the tempo for 8 beats. |
| 06 | **Sprint** | 5 rapid-fire Estimate rounds with a full summary at the end. |

## Features
- 3 difficulty levels (Easy / Normal / Hard)
- Web Audio API sound cues for every event
- Grading system: Perfect / Excellent / Good / Close / Miss
- Score, streak, average %, best % stats
- 7-round history dots
- Keyboard shortcuts: \Space\ to hold/tap, \Enter\ to submit, \R\ to replay

## Tech
Single HTML file — zero dependencies, zero build step, works offline.
